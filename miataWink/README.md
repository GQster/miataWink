# miataWink
DIY-ing a NA miata wink controler 


# Parts
- ESP32 (Im using the devkitV1, but really any MCU will work [arduino, teensy, etc])
- Low Level 5V 4 Channel Solid State Relay Module SSR G3MB-202P 240V 2A Output [link](https://www.aliexpress.us/item/3256805787569927.html?spm=a2g0o.order_list.order_list_main.5.341818021xawRT&gatewayAdapt=glo2usa)
- ~50ft 24 guage wire
- 12V to 5V converter
- screws, zipties, butt connectors, etc for actual car instalation
